Gerenciador Escolar

Sistema para fazer todo gerenciamento básico de uma instituição de ensino. É possível cadastrar alunos, endereços, professores, disciplinas, séries, turmas, avaliações e o diário de presença do aluno.

Gerenciador de Mercadorias

Sistema para fazer todo gerenciamento básico de mercadorias para lojas. Sistema para cadastrar, modificar, vender mercadorias e consultar relatório de venda.

Controle Financeiro

Sistema para adicionar e gerenciar dívidas do mês e dívidas de longo prazo, recebendo um resumo com todo valor gasto no mês e o total disponível com base na renda informada.
